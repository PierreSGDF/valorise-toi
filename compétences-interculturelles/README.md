# Compétences interculturelles

Les SGDF sont membres du **Scoutisme Mondial** (OMMS/WAGGGS), un réseau de 170 pays et 57 millions de scouts. Cette appartenance ouvre naturellement à l'international : camps européens, échanges de groupes, projets de solidarité internationale, Moot, Jamboree.

Ces expériences forgent une intelligence interculturelle rare, de plus en plus recherchée par les employeurs dans un monde globalisé.
